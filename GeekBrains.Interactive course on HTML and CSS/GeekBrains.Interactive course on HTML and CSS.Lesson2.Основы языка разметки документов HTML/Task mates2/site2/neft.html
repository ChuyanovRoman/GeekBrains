Картина Владимира Мигачёва "Нефть"

Описание товара:

Создана в 2008 г.

Характеристики товара:

  • Холст, смешанная техника.
  • Размер: 170 x 340 см

Подробное описание товара:

В работе «Нефть» 2008 года Владимир Мигачев изображает длинный грузовой состав, стремительно проходящий перед взглядом зрителя и удаляющийся, окутанный туманным, мрачным мороком.

Особенности товара:

Живопись Мигачева пастозна и сумрачна по колориту: тяжелое, свинцовое небо трудно отличимо от густой, вязкой массы, по которой проложена железная дорога.
Гигантские цистерны наполнены извлеченной из земных недр нефтью — природным ресурсом, ставшим для России символом благополучия и финансовой стабильности. Кажется, что темная субстанция сочится из цистерн, оставляя на холсте блестящие подтеки краски.
