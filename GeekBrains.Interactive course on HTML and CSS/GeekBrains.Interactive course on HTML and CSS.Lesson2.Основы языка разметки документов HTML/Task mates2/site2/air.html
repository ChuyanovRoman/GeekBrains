Картина Дмитрия Шорина "Пулково не принимает"

Описание товара:

Создана в 2008 г.

Характеристики товара:

  • Материалы: холст, масло;
  • Размер:100 x 199 см

Подробное описание товара:

В картине художник сталкивает изображения женщины и самолета в присущей ему резкой, эскизной манере.
Огромная машина по размеру оказывается сопоставимой с лежащей человеческой фигурой: самолет готовится к посадке, но «Пулково не принимает», и он вынужден, зависнув в нехарактерном для петербургского аэропорта синем небе, дожидаться команды. «Парящая» в этом же сияющем эфире женщина в смятении бросилась на кровать, в исступлении теребя холеными пальцами со свежим маникюром покрытое слоем косметики лицо и блестящие волосы.

Особенности товара:

Картина хранится в коллекции музея «Эрарта».
