Подробное описание товара

Стимулирует рост кораллов путём усиления предпочтительной для зооксантелл синей области спектра (так называемого актинического излучения)
Способствует красивой флуоресценции кораллов
В сочетании с JBL SOLAR MARIN DAY - идеальное освещение для морского аквариума
Длительный срок службы и надёжная работа
150% светоотдача по сравнению с обычными люминесцентными лампами с тем же спектром
При использовании отражателей (JBL SOLAR Reflect) количество света удваивается
Долгий срок службы и надёжность работы достигается специальным долговечным покрытием, вольфрамовыми электродами и инертным газом
Флуоресцентный эффект кораллов виден при выключенном дневном свете в свете только ламп JBL SOLAR OCEAN BLUE
Также подходит как сумеречное освещение для пресноводных аквариумов.
