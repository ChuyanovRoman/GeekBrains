подробное описание товара

Внутренний SSD-накопитель Samsung 860 EVO – обновленная модель, которая подходит для высокопроизводительных игровых и офисных компьютеров. Его мощный контроллер позволил довести скоростные характеристики до теоретического предела интерфейса SATA – 520/550 Мб/с при записи и чтении соответственно
Стабильная работа. Значительная часть свободных ячеек памяти может преобразовываться в одноуровневый кэш. Это решение ускоряет доступ к часто используемой информации, позволяя загружать большие файлы и запускать ресурсоемкие приложения за считанные секунды
Исключительная надежность. Накопитель построен на основе 64-уровневой памяти MLC NAND, которая успешно прошла длительное тестирование в лабораториях Samsung. Она выдерживает до 600 циклов полной перезаписи или ежедневное обновление трети контента на протяжении 5 лет
Простое управление. Вместе с SSD покупатель получает приложение Magician, позволяющее выполнять диагностику устройства, повышать его производительность, обновлять прошивку и ограничивать доступ к файлам путем шифрования
