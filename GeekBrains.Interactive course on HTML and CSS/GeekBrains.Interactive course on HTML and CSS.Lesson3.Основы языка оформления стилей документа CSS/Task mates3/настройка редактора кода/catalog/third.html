подробное описание товара

Внутренний SSD накопитель Samsung 850 PRO 2.5" SATA III 2TB надежно хранит информацию, а также значительно ускоряет работу с компьютером благодаря высокой скорости передачи данных.
Большой объем, высокая скорость. Благодаря новейшей технологии Samsung 3D V-NAND накопитель сочетает большой объем и великолепную надежность с непревзойденной скоростью передачи данных и при последовательном, и при случайном доступе к ним, заметно превосходя аналогичные модели других производителей
Еще больше скорости. Владелец накопителя может переключиться в режим Rapid, позволяющий еще больше увеличить скорость передачи данных за счет использования свободной оперативной памяти компьютера.
Невероятная долговечность. Ресурс записи накопителя настолько велик, что при записи на него 40 Гб ежедневно он будет выработан лишь через 10 лет.
Максимальная надежность. Благодаря динамической теплозащите накопитель никогда не выйдет из строя из-за перегрева, а 256-битное шифрование данных гарантируют конфиденциальность и безопасной информации, которая на нем хранится.
