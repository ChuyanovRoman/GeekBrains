Детское молочко для тела White Mandarin

Детское молочко для тела White Mandarin рекомендовано ведущими педиатрами страны.
Применять можно с самого рождения крохи. Кожа ребенка, особенно у новорожденного, очень нежная, склонна к высыпаниям и раздражениям, потери влаги, и первая задача мамы - устранить все проявления, которые могут привнести дискомфорт.
Применять детское молочко можно под памперс, легкая основа быстро впитывается в кожу.



Объем: 200мл / 220г
