Детский защитный крем от непогоды White Mandarin

Детский защитный крем White Mandarin был разработан с учетом нежной кожи ребенка, склонной к воспалению при любом внешнем воздействии, даже природном. Эта компактная баночка с богатым составом станет постоянным спутником на прогулках, ведь состав из растительных компонентов оберегает кожу от палящего солнышка, ветряной погоды и холодов.


Объем: 50мл / 50г
